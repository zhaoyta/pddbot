# Contributing

感谢贡献！完整说明（中文）见：**[md/contributing.md](md/contributing.md)**。

## 快速摘要

- Python **3.11**，推荐使用 **[uv](https://github.com/astral-sh/uv)** 安装依赖
- 提交前勿包含 `.env`、`storage_state.json`、数据库与密钥；运行 `uv run python -m llm._smoke_test` 做基础自检（适用时）
- 贡献代码即以 **[Apache License 2.0](LICENSE)** 授权（与仓库一致）

安全问题请勿公开贴细节，请先阅读 **[SECURITY.md](SECURITY.md)**。
